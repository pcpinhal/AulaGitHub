Passos ....

  • Instalar GIT

    • https://git-scm.com/downloads

  • Acesse o GitHUB , faça o login, clien no botao verde New project
  • Clique no botao verde code / local HTTPS e copie o endereço
  • va no VSCode, abra o terminal COMO BASH
  • digite o comando : git init
  • Crie um arquivo chamado README.MD e dentro dele comente sobre o seu projeto
  • agora ela ficou na pasta master.
  • agora digite: git add README.MD
  • para enviar todos os arquivos basta digita: git add .
  • ele acabou de enviar o arquivo para o git, otimo, entao vamos para o proximo passo
  • digite: git status
  • ele ira mostrar como esta a nosta lista de arquviso enviados e pendentes
  • digite: git commit -m "Enviando meu primeiro arquivo"
  • quanbdo vc cria um novo projeto no github ele jamostra os comandos....
  • assim sendo vamso ter q mudar tudo
  • git add . envia tudo
  • status verifica o q tem de modificaçã
  • git commit -m "msg que quiser" para explicxar o q mudou no arquivo
  • git push origin master(main) para enviar os arquivos para o github
  • git checkout -b "nova_acao"
  • ele faz com que eu crie uma nova area de desenvolvimento como um botao novo no site
  • crie o arquivos botao.html
  • adicione o arquivo ao projeto: git add .
